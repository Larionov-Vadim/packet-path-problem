Genetic Algorithm
-----
Solution of the finding the optimal path problem by the genetic algorithm method.

#### CSV file format
```
{fromNode},{toNode}
{Symmetric matrix adjacency}
```
See examples [input1.csv](./examples/input1.csv) and [input2.csv](./examples/input2.csv)

#### Requirements:
* Java 1.8
* Maven

#### Usage
```
./build.sh
./run.sh
```
